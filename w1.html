Week 1 | VCarve


The task for this week was to use VCarve to create our name plate. 'V-Carve' is an application that allows to cut parts on a CNC Router. This app has the ability to transform 2D designs to 3D designs, and calculate its toolpath. The most challenging part of this project was to set the machine up for use. Since it was my first using the machine, I was struggling to figure out which button to press. Although a bit overwhelming at first, the machine seemed to be relatively straightforward.



Process


To make using the machines an easier process, I will explain the steps I took: 1) Save your files as a "shopsabre inch.tap" file on a flashdrive 2) Go to the CNC machine 3) After you open up the Router, press "home" 4) Bring the magnet close to the left edge and "auto z" or "z-zero" 5) Then, bring the magnet to the left edge and "x-y zero" 6) You are done setting up the machine! :) Once you are done, open your file, and start printing!
